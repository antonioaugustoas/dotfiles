# 💻 Dotfiles

**Objetivo Central:** Suas configurações de ambiente como código. Automatização para subir máquinas em 5 minutos.
**Fronteira (O que DEVE ter):**
- Scripts de instalação (`setup.ps1`, `setup.sh`).
- Configurações exportadas do Windows/Linux (aliases, prompts).
- Preferências de ferramentas (VS Code `settings.json`, Git `.gitconfig`, perfis de Windows Terminal).

**Extrapolação (O que NÃO colocar aqui):**
- Projetos de estudo (Vai para `study-vault`).
- Anotações de atalhos e dicas de IDEs (Vai para `knowledge-vault`).
